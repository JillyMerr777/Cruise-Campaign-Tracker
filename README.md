Marketing Campaign Dashboard is a responsive, mobile-first internal tool designed for cruise line marketing teams to track, analyze, and optimize campaign performance across multiple guest touchpoints, including social media, email, website, and mobile app. Built as a single-page application using Vue 3, TypeScript, Vuetify, and Chart.js, the dashboard provides a real-time, high-level view of active and historical campaigns, surfacing key KPIs, budget pacing, channel performance, and actionable insights. It supports the full campaign lifecycle—from planning and creation to launch, monitoring, and optimization—while incorporating accessibility best practices, modern UI design, and realistic data to simulate a production-ready marketing intelligence platform.
